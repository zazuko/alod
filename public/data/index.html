Le domaine data.alod.ch constitue le point d'entrée de la description technique du projet aLOD. Il s'agit d'un prototype dont l'objectif est de servir de banc de test pour l'évaluation du potentiel de la technologie Linked Data (LD) dans le domaine des archives.

Les données elles-mêmes sont représentées par les domaines liés à chaque institution. (data.ge.alod.ch, data.ne.alod.ch, data.vs.alod.ch, data.staatsarchiv-bs.ch, data.alod.ch)

Ce site, mis à disposition par le projet aLOD, offre :
  • des données déréférençables par leurs URIs (par ex. http://data.ge.alod.ch/id/archivalresource/adl-j-125, http://data.staatsarchiv-bs.ch/id/archivalresource/CH-000027-1/pa-633c-a-12-fasc-2)
  • une interface technique SPARQL

Ouverture des métadonnées en "Utilisation libre" Open Data Swiss.

Les métadonnées mises à disposition dans le cadre du projet sont utilisables selon les termes de la licence Open Data Swiss Utilisation libre. Les données peuvent ainsi être librement copiées, modifiées et distribuées.
